# A7 Minima

A minimalist Artix-7 SoM in a credit-card form factor.

## Features

- XC7A100T-2 in CSG324 package
- 1GB DDR3 (32-bit, 800 Mbps)
- 32MB QSPI Flash
- On-board USB-to-JTAG + UART
- HDMI source connector
- 64 GPIOs compatible with perfboard
- 8 addressable RGB LEDs
- 2 single-color LEDs
- 3 slide switches
- 1 push button
